# IAIP Task-1 - Personal Portfolio Website

A responsive Personal Portfolio Website developed as part of the Internalpha Internship Program (IAIP Task-1). This project showcases my profile, skills, resume, and contact information in a clean and professional layout.

## 🚀 Features

- Responsive portfolio design
- Introduction/About Me section
- Skills section
- Projects showcase section
- Contact information
- Downloadable resume
- Profile image integration
- User-friendly interface

---

## 📂 Project Structure

```
IAIP-Task-1/
│
├── video/             # Project demonstration video
├── index.html         # Main portfolio webpage
├── style.css          # Styling and layout
├── image.jpeg         # Profile image

```
